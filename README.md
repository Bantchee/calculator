# Calculator
### Discription
A browser calculator app.

### Features
* Basic Arithmetic Functions
    * addition
    * subtraction
    * multiplication
    * division
* Operate Function 
    * takes an operator and 2 numbers, then calls an arithemtic function
* GUI
    * created and styled with HTML, CSS, and JavaScript
    * display
    * digit buttons, 0 - 9
    * arithmetic buttons
    * "equals" button
    * "clear" button
    * "Delete" button 
    * "Decimal" (.) button
    * "Power" (Maybe)
    * "(" (Maybe)
    * ")" (Maybe)
* UpdateDisplay Function
    * updates display when you click a button
    * store display value in JS variable (Array)

### Built With
* HTML
* CSS
* JavaScript
